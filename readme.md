# CyberGuard – ML-Based Network Threat Detection Dashboard

CyberGuard is a machine learning–based Intrusion Detection System (IDS) designed to monitor and analyze network traffic in real time.

## Features
- Real-time traffic monitoring
- Threat detection using ML
- Flask-based dashboard
- Alert logging system
- Live monitoring
- Confidence score prediction

## Technologies Used
- Python
- Flask
- Scikit-learn
- Scapy
- SQLite
- HTML/CSS/JavaScript

## Project Objective
To demonstrate how machine learning can be integrated with cybersecurity systems for network threat detection and monitoring.
